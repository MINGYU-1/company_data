# 📊 강원도 사업집계정보: 컬럼 정의서체 공간

이 문서는 `강원도_사업체_공간집계정보.csv` 파일의 각 컬럼이 무엇을 의미하는지 설명합니다.

| 컬럼명 (Column Name) | 의미 (Description) | 데이터 타입 (Type) | 비고 (Notes) |
| :--- | :--- | :--- | :--- |
| `index` | 행 인덱스 | Integer | 0부터 시작하는 단순 행 번호 |
| `UPLFT_XCNTS` | 좌상단 X좌표 | Float | 격자의 왼쪽 위 X좌표 |
| `UPLFT_YDNTS` | 좌상단 Y좌표 | Float | 격자의 왼쪽 위 Y좌표 |
| `LOWRGT_XCNTS` | 우하단 X좌표 | Float | 격자의 오른쪽 아래 X좌표 |
| `LOWRGT_YDNTS` | 우하단 Y좌표 | Float | 격자의 오른쪽 아래 Y좌표 |
| `ID` | 격자 ID | Integer | 각 격자(Grid)의 고유 ID |
| `BSNES_CNT` | 사업체 수 | Float | 해당 격자 내 사업체 총 수 |
| `LABRR_CNT` | 종사자 수 | Float | 해당 격자 내 총 종사자 수 |
| `LABRR_CNT_AVRG` | 평균 종사자 수 | Float | (종사자 수 / 사업체 수) |
| `SELNG_AMT` | 매출액 | Float | 해당 격자 내 총 매출액 |
| `SELNG_AMT_AVRG` | 평균 매출액 | Float | (매출액 / 사업체 수) |
| `SPCE_INFO` | 공간정보 | String | 격자의 POLYGON 좌표 (WKT 형식) |
